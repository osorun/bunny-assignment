# bunny-assignment
JavaScript object assignment - declaring a bunny object with name, age, and isHappy properties.

## Array Exercise - Bunny Names

**Objective:** Practice array methods (push, splice, unshift)

**What I Did:**
- Created an array called `bunnies` with six bunny names
- Added "Mario" to the end using `.push()`
- Removed "Lucy" from the array using `.splice()`
- Added "Luigi" to the beginning using `.unshift()`

**Final Array:**
['Luigi','Fluffy','Bugs-Bunny','Thunder','Snowball','Bella', 'Mario']

**Key Learnings:**
- Arrays start at index 0
- `.push()` adds to the end
- `.unshift()` adds to the beginning
- `.splice(index, 1)` removes an item at a specific position
